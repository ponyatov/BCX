bcx:MCU

особенности сборки для микроконтроллеров

  • bcx:MSP430

Для компиляции в составе прошивки с использованием IAR необходимо определить define-параметр компилятора BCX2C.

Так как IAR не имеет утилит типа GNU msp430-objcopy, а байт-код имеет небольшой размер, вполне работает метод конвертации в исходный код на Си.

cc -o b2c b2c.c bcx.c
bc2c FORTH.bc FORTH.c

Утилита b2c получает два параметра: файл с байткодом, и выходной .c

#include "bcx.h"
#ifdef BCX2C
BYTE M[Msz]={
0x03,0x12,0x00,0x00,0xFF,0x04,0x05,0x34,
0x12,0x05,0xAD,0xDE,0x05,0xAF,0xBE,0x05,
0x0D,0x00,0x00,0x01,0x04,0x00,
0};
UCELL Cp =0x0016;
#endif

Кроме инициализации M устанавливается указатель компиляции Cp
