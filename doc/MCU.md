# bcx:MCU  {#bcxmcu}
@brief особенности сборки для микроконтроллеров

* [bcxmsp430]

Для компиляции в составе прошивки с использованием IAR необходимо определить
define-параметр компилятора @ref BCX2C.

Так как IAR не имеет утилит типа GNU `msp430-objcopy`, а байт-код имеет
небольшой размер, вполне работает метод конвертации в исходный код на Си.

```
cc -o b2c b2c.c bcx.c
bc2c FORTH.bc FORTH.c

```

Утилита `b2c` получает два параметра: файл с байткодом, и выходной .c
